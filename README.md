## Getting Started
Visit the website [here](https://portfolio.kongsvinger-it.no/)

### Installing

Clone the repo:

```bash
git clone https://github.com/jonhavbra87/portfolio-1
```

### Running
Clone the repo, open in VSCode. Right click on the index.html and open in Live Server.
